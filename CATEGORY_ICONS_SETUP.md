# Category Icons Setup Guide

## Required Flaticon Icons

Download the following icons from Flaticon and save them in the `images/` folder with the specified names:

### 1. Phone Case Icon
- **Source**: https://www.flaticon.com/free-icons/phone-case (by surang)
- **Save as**: `images/icon-phone-case.png`

### 2. Phone Charger Icon
- **Source**: https://www.flaticon.com/free-icons/phone-charger (by Iconjam)
- **Save as**: `images/icon-phone-charger.png`

### 3. Smart Watch Icon
- **Source**: https://www.flaticon.com/free-icons/smart-watch (by Ehtisham Abid)
- **Save as**: `images/icon-smart-watch.png`

### 4. Accessories Icon
- **Source**: https://www.flaticon.com/free-icons/accessories (by HAJICON)
- **Save as**: `images/icon-accessories.png`

### 5. Device Icon
- **Source**: https://www.flaticon.com/free-icons/device (by xnimrodx)
- **Save as**: `images/icon-device.png`

## How to Download

1. Visit each Flaticon URL above
2. Click the download button
3. Choose PNG format
4. Save the file with the exact name specified above in the `images/` folder

## Icon Mapping

The system automatically maps category names to appropriate icons:

- **Phone Cases** → `icon-phone-case.png`
- **Chargers/Power/Charging** → `icon-phone-charger.png`
- **Smart Watch/Watches** → `icon-smart-watch.png`
- **Accessories/Audio/Headphones** → `icon-accessories.png`
- **Cables/Wires** → `icon-phone-charger.png`
- **Default/Other** → `icon-device.png`
